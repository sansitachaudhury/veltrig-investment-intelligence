## Overview

Veltrig is a beginner-focused investment intelligence platform designed to simplify stock analysis and market interpretation.

The platform combines forecasting, risk analysis, portfolio suggestions and educational guidance into a clean and accessible dashboard experience for new investors.

Instead of overwhelming users with technical financial data Veltrig focuses on clarity, risk awareness and right decision-making.

---

## Problem Statement

New investors often struggle with:

- Understanding stock volatility
- Interpreting market trends
- Managing investment risk
- Building balanced portfolios
- Filtering financial noise and hype

Most financial platforms are either too technical or too simplistic.

Veltrig aims to bridge that gap with beginner-friendly investment intelligence.

---

## Features

- Live NSE stock analysis
- AI-based trend forecasting
- Forecast confidence scoring
- Investment risk analysis
- Beginner-focused investment guidance
- Personalized investor insights
- Suggested portfolio allocation
- Interactive stock visualizations
- Educational investment tips
- Real-time market data integration

## Installation

Clone the repository:  
git clone https://github.com/sansitachaudhury/veltrig-investment-intelligence

Move into the project directory:  
cd veltrig-investment-intelligence

Install dependencies:  
pip install -r requirements.txt

Run the application:  
streamlit run app.py
